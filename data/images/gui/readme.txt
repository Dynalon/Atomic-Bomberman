explanation of the names
========================

t = top
b = bottom
l = left
r = right
m = middle

bor = border
chk = checkbox
cro = cross
spe = spinedit

sb = scrollbar

pls = plus
mns = minux

mo = mouse over
md = mouse down
hf = has focus
nf = not focused

example: mouse is over a button:
==> the bor_mo_**.tga is used.

that's it.
